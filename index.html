Avance dans le Livre, verset par verset

Trois voies, un même chemin : mémorisation (ḥifẓ), récitation (tilāwa) et compréhension (tafsīr). Choisis une voie, ouvre une sourate, marque chaque verset. Ta progression se synchronise sur un GitHub Gist privé.

À faire En cours Acquis
Clic = avancer le statut (à faire → en cours → acquis → à faire). Les trois points sous un verset indiquent son état dans les autres voies.
